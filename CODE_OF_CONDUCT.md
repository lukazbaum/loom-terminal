# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its code of
conduct.

In short:

- Be respectful. Assume good faith. Disagree on the substance, not the person.
- Keep public spaces (issues, PRs, discussions) focused on the project.
- Help newcomers; everyone was new once.

Behaviour that makes participation feel unwelcome to others — personal
attacks, harassment, or sustained disruption — is not acceptable in any
project space.

## Reporting

If you experience or witness unacceptable behaviour, report it privately to
**lukas@lukasbaum.dev**. Reports will be reviewed and handled with discretion.

## Scope

This Code of Conduct applies in all project spaces (the repo, issues, PRs,
discussions) and when an individual is representing the project in public.
